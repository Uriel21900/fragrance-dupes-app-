# Fragrance Dupes

A web application to discover affordable alternatives to popular fragrances.

## Features

- Browse popular fragrances
- Click on any fragrance to see its dupes/clones
- Compare original prices to dupe prices
- Search fragrances by name or brand

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Adding New Fragrances

Edit `src/App.tsx` and add new entries to the `fragrances` array following the existing structure.
